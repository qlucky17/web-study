## 2. 标题：#
```js
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题
```

## 3. 字体样式：
```js
加粗：**文字**
高亮：==文字==
删除：~~文字~~
斜体： *文字*
粗斜体：***文字***
```
**等不到天黑**

==等不到天黑==
~~等不到天黑~~
*等不到天黑*

***等不到天黑***
